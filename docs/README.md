# 구현 기능 목록
## 1. 입력
- 사용자가 콘솔에 숫자를 입력하고 그 값을 받는다.
- 사용자가 게임을 계속 진행할 지 여부를 받는다.

## 2. 출력
- 시작 메시지를 출력한다.
- 숫자 입력 요청 메시지를 출력한다.
- 입력한 숫자에 대한 결과 메시지를 출력한다.
- 정답을 맞췄을 때의 메시지를 출력한다.
- 게임 계속 진행 여부 메시지를 출력한다.

## 3. 랜덤 숫자 생성
- 각 자리의 숫자를 생성한다.
- 각 자리의 숫자를 리스트로 변환한다.

## 4. 사용자 입력값 검증
- 사용자가 입력한 값을 리스트로 변환한다.
- 3자리 여부를 확인한다.
- 각 자리의 숫자 여부를 확인한다.
- 조건에 맞지 않는다면 예외를 발생시킨다.

## 5. 판단
- 입력 값에 대한 볼 개수를 반환한다.
- 입력 값에 대한 스트라이크 개수를 반환한다.
- 두 개의 정보를 담아 반환한다.
